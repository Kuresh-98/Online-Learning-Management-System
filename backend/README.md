# LMS Backend API

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Test the API:
   - Open browser: http://localhost:5000
   - Or use Thunder Client/Postman

## Available Scripts

- `npm start` - Run production server
- `npm run dev` - Run development server with auto-reload
